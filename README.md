## ⚡ Installation & Setup

Clone the repository:

```bash
git clone https://github.com/manavpal-dev/manav-portfolio.git
```

Navigate to the project folder:

```bash
cd manav-portfolio
```

Install dependencies:

```bash
npm install
```

Start the development server:

```bash
npm run dev
```
